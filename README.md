# Global Financial Inclusion Analysis (Power BI)

## Overview
This group project analyses global financial inclusion patterns using the World Bank Global Findex 2025 dataset.
The focus is on regional, income-level, and digital finance disparities.

## Tools
- Power BI
- Power Query
- DAX
- World Bank Global Findex Dataset (2011–2024)

## Key Skills Demonstrated
- Data cleaning & transformation
- Business intelligence dashboard design
- Socioeconomic analysis
- Data storytelling & visual analytics

## My Contribution
- Data cleaning and validation in Power Query
- DAX measures for digital inclusion metrics
- Interpretation of regional and income-level trends
- Academic report writing and visual explanation

## Outputs
- Interactive Power BI dashboard
- 27-page analytical report (PDF)
